🔄 ПРИНЦИП 1: ЗАПИТ → ВІДПОВІДЬ (Request-Response)

5
Браузер отримує відповідь і відображає вам сторінку! ✅

📝 ПРИКЛАД 1: СИСТЕМА РЕЄСТРАЦІЇ ТА ВХОДУ

🔑 Як це працює (крок за кроком):

1
Користувач вводить email + пароль і натискає «Увійти»
2
Браузер надсилає дані на сервер (POST-запит)
3
Сервер перевіряє email у базі даних
4
Порівнює хеш пароля → якщо збіг — видає токен
5
Браузер зберігає токен і відкриває особистий кабінет ✅

🐍 Код на Python (Flask):

from flask import Flask, request, jsonify
from werkzeug.security import check_password_hash

app = Flask(__name__)

@app.route('/login', methods=['POST'])
def login():
    data = request.json
    user = db.find_user(data['email'])

    if user and check_password_hash(
        user.password, data['password']
    ):
        token = generate_token(user.id)
        return jsonify({'token': token})

    return jsonify({'error': 'Невірні дані'}), 401

🛒 ПРИКЛАД 2: ІНТЕРНЕТ-МАГАЗИН

⚙️ Що робить сервер:

📦
Каталог товарів — зберігає тисячі товарів з цінами і фото в БД
🔍
Пошук — обробляє запит і знаходить товари в мілісекундах

Кошик — запам'ятовує обрані товари для кожного користувача
💳
Оплата — безпечно з'єднується з платіжним шлюзом (Stripe, LiqPay)

🟢 Код на Node.js (Express):

const express = require('express');
const app = express();

// Отримати список товарів
app.get('/products', async (req, res) => {
  const { search, category } = req.query;
  const products = await db.query(
    `SELECT * FROM products
     WHERE name LIKE ?
       AND category = ?`,
    [`%${search}%`, category]
  );
  res.json(products);
});

// Додати товар до кошика
app.post('/cart/add', async (req, res) => {
  const { userId, productId } = req.body;
  await db.addToCart(userId, productId);
  res.json({ success: true });
});

🐍 ПРОСТИЙ СЕРВЕР НА PYTHON (Flask)

📄 Код сервера (app.py):

from flask import Flask, jsonify

app = Flask(__name__)

# Зберігаємо студентів в пам'яті
students = [
    {"id": 1, "name": "Іван", "grade": 90},
    {"id": 2, "name": "Оля",  "grade": 85},
]

# GET /students — повернути всіх студентів
@app.route('/students')
def get_students():
    return jsonify(students)

# GET /students/1 — студент за ID
@app.route('/students/<int:id>')
def get_student(id):
    student = next(
        (s for s in students if s['id'] == id),
        None
    )
    return jsonify(student) if student \
           else ('Not Found', 404)

if __name__ == '__main__':
    app.run(debug=True)

🚀 Запуск та результат:
