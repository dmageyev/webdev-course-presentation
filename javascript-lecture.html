⚡ ПРОБЛЕМА: КОД БЕЗ jQuery VS З jQuery

Задача: при кліку на кнопку — сховати її та змінити колір

❌ Vanilla JS — 10+ рядків:

document.addEventListener('DOMContentLoaded',
  function() {
    var buttons =
      document.getElementsByClassName('btn');
    for (var i = 0; i < buttons.length; i++) {
      buttons[i].addEventListener('click',
        function() {
          this.style.backgroundColor = 'red';
          this.style.display = 'none';
        }
      );
    }
  }
);

✅ jQuery — 3 рядки:

$(function() {
  $('.btn').click(function() {
    $(this).css('background', 'red').hide();
  });
});

І ОБИДВА РОБЛЯТЬ ТЕ САМЕ! 🎯

🔌 ЯК ПІДКЛЮЧИТИ jQuery

⭐ Спосіб 1: CDN (рекомендовано)

Підключення через Інтернет — нічого не треба завантажувати

<!-- Додайте перед </body> -->
<script
  src="https://code.jquery.com/jquery-3.7.0.min.js">
</script>

<!-- Ваш код після підключення -->
<script>
  $(function() {
    // Ваш jQuery-код тут
  });
</script>

📥 Спосіб 2: Локальний файл

Завантажити файл jquery.min.js на власний сервер

<!-- Файл поруч з вашим HTML -->
<script src="js/jquery.min.js"></script>

<script>
  $(function() {
    // Ваш jQuery-код тут
  });
</script>

Для більшості задач — CDN простіше ✅

🔌 Аналогія: CDN — як розетка в стіні. Підключив один кабель — і все обладнання працює!

🔍 ПРИНЦИП 1: СЕЛЕКТОР $()

$ = псевдонім jQuery() — функція для пошуку елементів на сторінці

Синтаксис Що шукає CSS-аналог
$('#logo') Елемент з id="logo" #logo { ... }
$('.btn') Всі елементи з class="btn" .btn { ... }
$('p') Всі параграфи <p> p { ... }
$('a.active') Посилання з класом "active" a.active { ... }
$('ul li') Всі <li> всередині <ul> ul li { ... }

💡 Аналогія: $('#title') — як сказати другу:
"Знайди мені людину на ім'я Title у цій кімнаті"

⏳ ПРИНЦИП 2: $(document).ready()

Запускає код лише після того, як вся сторінка завантажена

📝 Повна форма:

$(document).ready(function() {

  // Ваш jQuery-код тут.
  // Виконується тільки коли
  // HTML-сторінка повністю завантажена

  $('#btn').click(function() {
    alert('Готово!');
  });

});

⚡ Скорочена форма (те саме):

$(function() {

  // Ідентичний результат!
  // Більшість розробників
  // використовують саме цю форму.

  $('#btn').click(function() {
    alert('Готово!');
  });

});

☕ Аналогія: Спочатку повністю завантажити сторінку — потім починати з нею взаємодіяти. Як чекати, поки кава заварилась!

🔗 ПРИНЦИП 3: ЛАНЦЮЖКИ МЕТОДІВ (CHAINING)

Методи можна об'єднувати в ланцюжок — кожен виконується з результатом попереднього

❌ Без ланцюжка (4 окремі рядки):

$('#box').css('color', 'white');
$('#box').css('background', '#4285F4');
$('#box').html('Привіт, jQuery!');
$('#box').fadeIn(600);

✅ З ланцюжком (1 рядок):

$('#box')
  .css('color', 'white')
  .css('background', '#4285F4')
  .html('Привіт, jQuery!')
  .fadeIn(600);

🏭 Аналогія: Конвеєр на заводі — деталь проходить через фарбування → шліфування → пакування → відправлення, не виходячи з лінії

📝 МАНІПУЛЯЦІЯ ВМІСТОМ

КОД:

// Встановити HTML-вміст (з тегами):
$('#title').html('<b>Новий заголовок!</b>');

// Встановити лише текст (без тегів):
$('#subtitle').text('Простий текст');

// Отримати поточний текст:
var current = $('#subtitle').text();
console.log(current); // "Простий текст"

// Поле вводу (input) — .val():
$('#email').val('user@example.com');

// Отримати значення з поля:
var email = $('#email').val();
console.log(email); // "user@example.com"

МЕТОДИ:

.html()
Змінює HTML-вміст елемента
(може містити теги)
.text()
Змінює тільки текст
(теги ігноруються)
.val()
Значення поля форми
(input, textarea, select)

🎨 СТИЛІ ТА CSS-КЛАСИ

КОД:

// Встановити один стиль:
$('.box').css('background-color', '#4285F4');

// Встановити кілька стилів одразу:
$('.box').css({
  'background': '#4285F4',
  'color': 'white',
  'border-radius': '10px'
});

// Додати CSS-клас:
$('.btn').addClass('active');

// Видалити CSS-клас:
$('.btn').removeClass('active');

// Перемкнути (є — видали, нема — додай):
$('.btn').toggleClass('active');

МЕТОДИ:

.css()
Встановлює inline-стилі
.addClass()
Додає CSS-клас до елемента
.removeClass()
Видаляє CSS-клас
.toggleClass()
⭐ Додає або видаляє клас
(залежно від наявності)

🖱️ ОБРОБКА ПОДІЙ

Подія — це будь-яка дія користувача: клік, наведення, введення тексту

🖱️ Клік по кнопці:

$('#btn').click(function() {
  alert('Кнопку натиснуто!');
  $(this).text('Натиснуто ✅');
});

🏷️ Наведення миші:

$('.card').hover(
  // Миша НАЇХАЛА:
  function() {
    $(this).addClass('highlighted');
  },
  // Миша ПІШЛА:
  function() {
    $(this).removeClass('highlighted');
  }
);

⌨️ Введення тексту:

$('#search').on('input', function() {
  var query = $(this).val();
  $('#result').text('Пошук: ' + query);
});

📋 Відправка форми:

$('#form').on('submit', function(e) {
  e.preventDefault(); // Зупинити перезавантаження
  var name = $('#name').val();
  alert('Дякуємо, ' + name + '!');
});

✨ АНІМАЦІЇ

jQuery надає готові анімації — жодного CSS не потрібно!

👁️ Показати / Сховати:

$('#box').show();         // Показати
$('#box').hide();         // Сховати
$('#box').toggle();       // Перемкнути стан

🌅 Плавне зникнення:

$('#box').fadeIn(500);    // З'явитись за 0.5с
$('#box').fadeOut(500);   // Зникнути за 0.5с
$('#box').fadeToggle(300);// Перемкнути

📖 Розгортання:

$('#menu').slideDown(400);   // Розкрити вниз
$('#menu').slideUp(400);     // Закрити вгору
$('#menu').slideToggle(300); // Перемкнути

🎨 Довільна анімація:

$('#box').animate({
  width: '300px',
  opacity: 0.5,
  marginLeft: '50px'
}, 800); // тривалість 0.8 сек

💡 Число в дужках — тривалість у мілісекундах (1000 мс = 1 секунда)

🖼️ РЕАЛЬНИЙ ПРИКЛАД: ГАЛЕРЕЯ ЗОБРАЖЕНЬ

Клік на мініатюру → відображення великого фото — лише 5 рядків коду!

КОД:

<!-- HTML структура -->
<div class="gallery">
  <img class="thumb"
    src="photo1_small.jpg"
    data-full="photo1_large.jpg">
  <img class="thumb"
    src="photo2_small.jpg"
    data-full="photo2_large.jpg">
</div>

<img id="preview" src="" style="display:none">
// jQuery:
$('.thumb').click(function() {
  var bigSrc = $(this).data('full');
  $('#preview')
    .attr('src', bigSrc)
    .fadeIn(400);
});

ПРИНЦИП РОБОТИ:

1. $('.thumb').click()
— Слухаємо клік на будь-якій мініатюрі
2. $(this).data('full')
— Беремо адресу великого фото з атрибута data-full
3. .attr('src', bigSrc)
— Встановлюємо нове зображення
4. .fadeIn(400)
— Плавно показуємо великий preview

❌ НЕПРАВИЛЬНО — чому не A і C?

⚖️ jQuery vs СУЧАСНИЙ JAVASCRIPT

jQuery — чудовий інструмент, але потрібно розуміти, коли його варто використовувати

✅ ВИКОРИСТОВУЙТЕ jQuery, коли:

  • 🔧 Працюєте з legacy-кодом (старим)
  • 🌐 Проекти на WordPress / Bootstrap
  • 🚀 Швидке прототипування без фреймворків
  • 🏛️ Підтримка старих браузерів обов'язкова
  • 📚 Вивчаєте DOM і хочете простіший старт

⚠️ ПОДУМАЙТЕ ДВІЧІ, коли:

  • ⚛️ Будуєте SPA на React / Vue / Angular
  • ⚡ Критична продуктивність (завантаження файлу)
  • 🧠 Вивчаєте сучасний ES6+ з нуля
  • 📱 Мобільний додаток (нативний)
  • 🛠️ Простий скрипт з 1–2 DOM-операціями

🎓 Висновок для початківців: jQuery — відмінний старт. Зрозумівши jQuery, легше освоїти сучасні фреймворки!
