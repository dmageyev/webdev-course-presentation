📜 КОРОТКА ІСТОРІЯ JAVASCRIPT

1995
Бренден Ейх (Netscape) створив мову всього за 10 днів. Спочатку називалась Mocha, потім LiveScript, і нарешті — JavaScript.
1997
Стандартизація: з'явився стандарт ECMAScript — офіційна специфікація мови.
2008
Google випустив браузер Chrome із V8-двигуном — JS став у десятки разів швидшим.
2009
Народився Node.js — тепер JS можна запускати на сервері, а не лише у браузері.
Сьогодні
JavaScript — найпопулярніша мова програмування у світі (рейтинг Stack Overflow 2023–2024).

🔍 JS-двигун — «перекладач»

Переводить JavaScript-код у машинні команди, які розуміє процесор.
Chrome → V8 | Firefox → SpiderMonkey | Safari → JavaScriptCore

⚡ Інтерпретована мова

JS не потребує компіляції перед запуском — він виконується рядок за рядком у реальному часі. Це спрощує розробку та налагодження.

💡 Аналогія: JS-двигун — як синхронний перекладач на міжнародній конференції: читає вашу мову й одразу «говорить» нею до комп'ютера.

🌳 DOM — «МОДЕЛЬ СТОРІНКИ»

DOM = Document Object Model

Браузер «перетворює» HTML-сторінку на дерево об'єктів. JS може знаходити будь-який елемент цього дерева й змінювати його.

💡 Аналогія

DOM — це як схема будівлі. JS — це будівельник, який за цією схемою може відкрити двері, пофарбувати стіну або прибудувати кімнату — прямо під час вашого перебування всередині. 

// Знайти елемент на сторінці
const кнопка = document.getElementById('myBtn');

// Змінити текст заголовку
const заголовок = document.querySelector('h1');
заголовок.textContent = 'Новий текст!';

// Змінити колір фону
document.body.style.backgroundColor = 'lightblue';

// Приховати елемент
кнопка.style.display = 'none';

КЛЮЧОВІ КОНЦЕПЦІЇ МОВИ

📦 ЗМІННІ — ЗБЕРІГАННЯ ДАНИХ

Змінна = іменована «коробка» для даних

Уявіть шафу з підписаними ящиками. У кожному ящику зберігається якесь значення. Ви можете покласти щось нове або забрати з будь-якого ящика.

Типи даних у JS:

  • 🔢 Числа: 42, 3.14
  • 📝 Рядки: "Привіт, світе!"
  • ✅ Булеві: true / false
  • 📋 Масиви: ["яблуко", "груша"]
  • 🗂️ Об'єкти: { ім'я: "Іван", вік: 20 }
// let — змінна, яку можна змінити
let ім'я = "Оксана";
let вік = 21;
let студент = true;

// const — константа (не змінюється)
const університет = "КПІ";

// Використання
console.log("Привіт, " + ім'я + "!");
// → "Привіт, Оксана!"

вік = 22; // змінили значення
console.log(вік); // → 22

🔧 ФУНКЦІЇ — НАБОРИ КОМАНД

Функція = «рецепт» або «інструкція»

Набір команд, який можна виконати скільки завгодно разів, просто назвавши його.
Як рецепт: написав один раз → готуєш щоразу.

Навіщо функції?

  • ♻️ Повторне використання коду
  • 📦 Організація логіки
  • 🔍 Легше знаходити помилки
  • 🤝 Командна розробка
// Оголошення функції
function привітати(ім'я) {
  return "Привіт, " + ім'я + "! 👋";
}

// Виклик функції
console.log(привітати("Марія"));
// → "Привіт, Марія! 👋"

console.log(привітати("Андрій"));
// → "Привіт, Андрій! 👋"

// Функція з обчисленням
function площаКімнати(ширина, довжина) {
  return ширина * довжина;
}

console.log(площаКімнати(4, 5)); // → 20

🔀 УМОВИ ТА 🔁 ЦИКЛИ

🔀 Умови (if / else)

let оцінка = 85;

if (оцінка >= 90) {
  console.log("Відмінно! 🌟");
} else if (оцінка >= 75) {
  console.log("Добре! 👍");
} else {
  console.log("Треба підтягнутись 📚");
}
// → "Добре! 👍"

💡 Аналогія: «Якщо температура > 25°C — йдемо на пляж, інакше — залишаємося вдома»

🔁 Цикли (for)

// Надрукувати числа від 1 до 5
for (let i = 1; i <= 5; i++) {
  console.log("Крок " + i);
}
// → Крок 1
// → Крок 2
// → ... Крок 5

// Обробка списку студентів
let студенти = ["Оля", "Іван", "Юля"];
for (let студент of студенти) {
  console.log("Привіт, " + студент + "!");
}

💡 Аналогія: «Пройти всю чергу та привітатися з кожною людиною»

🖱️ ПОДІЇ (EVENTS) — «РЕАКЦІЯ» НА ДІЇ

Подія — це сигнал: «щось сталося»

  • 🖱️ Клік мишкою на кнопку
  • ⌨️ Натискання клавіші
  • 📝 Введення тексту у поле
  • 📜 Прокручування сторінки
  • 🖼️ Завантаження зображення
  • ⏱️ Таймер спрацював

💡 JS «слухає» події та запускає потрібний код у відповідь — без перезавантаження сторінки!

// Знаходимо кнопку на сторінці
const кнопка = document.getElementById('myBtn');

// Призначаємо реакцію на клік
кнопка.addEventListener('click', function() {
  alert('Кнопку натиснуто! 🎉');
});

// Реакція на введення тексту
const поле = document.getElementById('input');
поле.addEventListener('input', function() {
  console.log('Ви ввели: ' + поле.value);
});

// Реакція на прокручування
window.addEventListener('scroll', function() {
  console.log('Позиція: ' + window.scrollY);
});

⏱️ ПРИКЛАД 3: ДИНАМІЧНЕ ОНОВЛЕННЯ ДАНИХ

Годинник, що оновлюється кожну секунду:

function оновитиЧас() {
  const зараз = new Date();

  const години = зараз.getHours()
    .toString().padStart(2, '0');
  const хвилини = зараз.getMinutes()
    .toString().padStart(2, '0');
  const секунди = зараз.getSeconds()
    .toString().padStart(2, '0');

  document.getElementById('clock').textContent =
    `${години}:${хвилини}:${секунди}`;
}

// Виклик кожну секунду (1000 мс)
setInterval(оновитиЧас, 1000);

Живий приклад:

00:00:00

Оновлюється автоматично кожну секунду

Де це застосовується?
Відлік часу до події, live-оновлення курсів валют, відображення статусу доставки, онлайн-моніторинг.

🌐 ПРИКЛАД 4: ОТРИМАННЯ ДАНИХ З ІНТЕРНЕТУ (API)

Що таке API?

API (Application Programming Interface) — це «меню» сервісу, де описано, які дані можна замовити.

💡 Аналогія: офіціант передає ваше замовлення на кухню (сервер) і повертає страву (дані).

Реальні приклади API-запитів:

  • 🌤️ Погода на сайті
  • 💱 Курс валют
  • 🗺️ Дані для карт
  • 📰 Стрічка новин
  • 🎵 Інформація про треки

Запит до API погоди:

// fetch — вбудована функція JS для запитів
async function отриматиПогоду(місто) {
  const відповідь = await fetch(
    `https://api.weather.com?city=${місто}`
  );

  const дані = await відповідь.json();

  document.getElementById('weather').innerHTML = `
    
${дані.city}

    
🌡️ ${дані.temperature}°C

    
💧 Вологість: ${дані.humidity}%

    
${дані.description}

  `;
}

⚙️ NODE.JS — JAVASCRIPT НА СЕРВЕРІ

До 2009 року: JS тільки у браузері

Після Node.js: JS скрізь 🌍

Node.js дозволяє запускати JavaScript-код на сервері — там, де раніше використовувалися Java, Python або PHP.

Що будують на Node.js:

  • 🌐 REST API та веб-сервери
  • 💬 Чати у реальному часі
  • 🔐 Системи авторизації
  • 📊 Обробка великих даних
  • 🤖 Боти для Telegram / Discord

Простий веб-сервер на Node.js:

const http = require('http');

const сервер = http.createServer((запит, відповідь) => {
  відповідь.writeHead(200, {
    'Content-Type': 'text/html; charset=utf-8'
  });

  відповідь.end('
Привіт з Node.js!
');
});

// Запускаємо сервер на порту 3000
сервер.listen(3000, () => {
  console.log('Сервер запущено! 🚀');
  console.log('Відкрийте http://localhost:3000');
});
