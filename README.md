- **Тема:** Що таке вебсайт? Перша HTML-сторінка
- **Тривалість:** 60 хвилин
- **Слайдів:** 25
- **Формат:** Онлайн (Google Meet)
- **Цільова аудиторія:** Діти 11-13 років

## 👨‍🏫 Для викладача

### Перед заняттям
1. Відкрийте презентацію заздалегідь
2. Натисніть `S` для режиму доповідача
3. Перевірте, що таймер працює (слайд 20)
4. Перевірте, що конфеті з'являється (слайди 15, 25)

### Під час заняття
- **Слайд 14** - дайте дітям 20 секунд на відповідь у чаті
- **Слайд 18** - переконайтеся, що всі бачать результат у CodePen
- **Слайд 19** - диктуйте код ПОВІЛЬНО, робіть паузи
- **Слайд 20** - таймер запуститься автоматично, вимагайте розминку!
- **Слайд 21** - дайте повні 15 хвилин на творче завдання

### Режим доповідача (клавіша S)
- Бачите поточний та наступний слайди
- Таймер заняття
- Нотатки (якщо додані в майбутньому)

### Корисні поради
- Використовуйте повноекранний режим (F)
- При технічних проблемах натисніть Esc для огляду
- Можна перейти на будь-який слайд натисканням номера + Enter
