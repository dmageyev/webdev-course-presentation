😩 ПРОБЛЕМА: JAVASCRIPT БЕЗ JQUERY

❌ Чистий JS (до 2006 року)

Різні браузери (IE, Firefox, Opera) підтримували різний синтаксис. Один і той самий код доводилось писати кілька разів для кожного браузера.

❌ Навіть проста задача — до 15 рядків:

// Приховати елемент — "по-старому"
var el = document.getElementById('panel');

if (el.style) {
  if (el.style.display !== 'none') {
    el.style.display = 'none';
  } else {
    el.style.display = '';
  }
}

// Додати обробник кліку (з підтримкою IE8)
if (el.addEventListener) {
  el.addEventListener('click', handler);
} else if (el.attachEvent) {
  el.attachEvent('onclick', handler);
}

✅ З jQuery — один рядок

jQuery бере на себе всю сумісність браузерів. Розробник пише лаконічний, зрозумілий код — і він працює скрізь.

✅ Той самий результат — 1 рядок:

// Перемкнути видимість
$('#panel').toggle();

// Додати обробник кліку
$('#panel').click(handler);

// Анімоване приховання
$('#panel').slideUp(400);

// Змінити текст та колір
$('#panel')
  .text('Оновлено!')
  .css('color', 'green');

💡 Девіз jQuery: "Write Less, Do More" — «Пиши менше, роби більше». Це і є головна мета бібліотеки.

🔷 ЩО ТАКЕ JQUERY?

jQuery — це бібліотека JavaScript

Бібліотека = готовий набір функцій, які хтось написав за вас.

💡 Уявіть конструктор LEGO: замість того щоб ліпити кожну деталь із пластику, ви просто обираєте потрібний блок і будуєте.

jQuery спрощує:

  • 🌳 Пошук і зміну елементів сторінки (DOM)
  • 🖱️ Обробку подій (кліки, натискання, наведення)
  • ✨ Анімацію та візуальні ефекти
  • 🌐 AJAX-запити до серверів
  • 🧩 Підключення плагінів одним рядком

📦 Розмір: ~30 КБ (мінімальна версія) — менший за одну фотографію.

Аналогія: jQuery як «пульт керування»

Без пульта: вставати, підходити до телевізора, вручну натискати кнопки.

З пультом: одна кнопка — і зміна каналу з дивана.

jQuery — це «пульт» для управління вебсторінкою замість «ходіння до телевізора» через довгий код.

Факти:

  • 🗓️ Вийшла у 2006 році (Джон Резіг)
  • 🌍 Використовується на ~76% з топ-10M сайтів
  • 🏢 Google, Microsoft, Amazon, BBC — всі використовують jQuery
  • 📥 Завантажується ~4 млрд разів/місяць

📜 КОРОТКА ІСТОРІЯ JQUERY

2006
Джон Резіг представив jQuery на BarCamp NYC. Перша версія розв'язала проблему крос-браузерної сумісності.
2007–09
Стрімкий ріст популярності. Google та Microsoft оголосили офіційну підтримку jQuery.
2011
Вийшов jQuery Mobile — спеціальна версія для мобільних сайтів.
2013
jQuery 2.0 відмовилась від підтримки застарілого IE 6/7/8. Код став легшим і швидшим.
2016
jQuery 3.0 — підтримка Promises, покращена продуктивність, нові методи.
Сьогодні
jQuery 3.7+ — активна розробка триває. Бібліотека залишається найпопулярнішою у світі, хоч і поступається фреймворкам у нових SPA-проектах.

💡 Факт: jQuery пережила всі «модні» технології початку 2010-х і досі активно використовується. Це свідчення її надійності та простоти.

🔌 ЯК ПІДКЛЮЧИТИ JQUERY?

Спосіб 1: CDN (рекомендовано)

Один рядок у HTML — і jQuery готова до роботи. Браузер завантажує файл із мережі доставки контенту.

<!-- Додати перед закриваючим </body> -->
<script src="https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.min.js">
</script>

<!-- АБО через Google CDN -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js">
</script>

Спосіб 2: npm (для розробників)

npm install jquery

⚠️ Важливий порядок підключення!

jQuery має бути підключена до вашого коду — інакше браузер не знатиме що таке $.

<!DOCTYPE html>
<html>
<head>
  <title>Мій сайт</title>
</head>
<body>

  <h1 id="title">Привіт!</h1>
  <button id="btn">Натисни мене</button>

  <!-- 1. Спочатку підключаємо jQuery -->
  <script src="jquery.min.js"></script>

  <!-- 2. Потім — наш код -->
  <script src="main.js"></script>

</body>
</html>

💲 ОСНОВНИЙ СИНТАКСИС: $()

Символ $ — серце jQuery

$() — це функція, що знаходить елементи на сторінці за CSS-селектором. Якщо знайомі зі звичайними CSS-селекторами — ви вже знаєте jQuery! 

// CSS-селектори — ті самі що в стилях!

// За ID (#)
$('#header')        → <div id="header">

// За класом (.)
$('.card')          → всі елементи з класом card

// За тегом
$('p')              → всі абзаци

// Складні селектори
$('ul li.active')   → <li class="active"> всередині <ul>
$('#form input')    → всі поля вводу у формі
$('button:first')   → перша кнопка на сторінці

// Знайти нащадків
$('#menu').find('a')  → всі посилання в #menu

📄 $(document).ready() — «ЧЕКАТИ СТОРІНКУ»

Навіщо це потрібно?

JavaScript може виконатись до того, як HTML-елементи з'явились на сторінці. Тоді $('#btn') нічого не знайде.

$(document).ready() каже jQuery: «Зачекай, поки вся сторінка завантажиться, і тільки тоді виконай код».

💡 Аналогія: Як не можна прикрасити кімнату, поки вона ще будується. Спочатку — стіни та двері, потім — декор.

// ❌ Помилка — код виконується до появи кнопки
$('#btn').click(function() {
  alert('Клік!');
});
// → Кнопка ще не існує в DOM!

// ✅ Правильно — чекаємо завантаження сторінки
$(document).ready(function() {
  $('#btn').click(function() {
    alert('Клік!');
  });
});

// ✅ Скорочений запис (те саме, але коротше)
$(function() {
  $('#btn').click(function() {
    alert('Клік!');
  });
  
  // Весь jQuery-код сайту пишеться тут ↑
});

🔗 CHAINING — ЛАНЦЮГОВИЙ ВИКЛИК

Ланцюговий виклик = методи через крапку

Кожен метод jQuery повертає той самий об'єкт, тому можна одразу викликати наступний метод.

💡 Як наказувати роботу одному виконавцеві послідовно: «знайди Олю → переодягни її → посміхнись їй → сфотографуй».

Переваги chaining:

  • 📝 Менше коду — елемент шукається один раз
  • ⚡ Швидше — немає повторних звернень до DOM
  • 👀 Зрозуміліше — всі дії над одним елементом разом
// ❌ БЕЗ chaining — шукаємо елемент 4 рази
$('#box').addClass('active');
$('#box').css('color', 'white');
$('#box').text('Готово!');
$('#box').fadeIn(300);

// ✅ З chaining — шукаємо 1 раз
$('#box')
  .addClass('active')    // 1. Додаємо клас
  .css('color', 'white') // 2. Змінюємо колір
  .text('Готово!')       // 3. Оновлюємо текст
  .fadeIn(300);          // 4. Показуємо з анімацією

// Ще один приклад — обробка форми
$('#loginForm')
  .find('input')         // знайти всі поля
  .val('')               // очистити значення
  .removeClass('error')  // зняти клас помилки
  .end()                 // повернутись до #loginForm
  .hide();               // сховати форму

🌐 ЩО ТАКЕ AJAX?

❌ Без AJAX — перезавантаження всієї сторінки

Натиснули кнопку «Лайк» → сторінка повністю оновлюється → втрата позиції прокручування → затримка 1–3 секунди. Дратівливо!

✅ З AJAX — миттєве оновлення частини сторінки

Натиснули «Лайк» → jQuery відправляє маленький запит у фоні → отримує відповідь → оновлює тільки лічильник лайків. Миттєво!

💡 AJAX = Asynchronous JavaScript and XML.
Де ви зустрічали: пошук Google (результати з'являються під час введення), Instagram (нескінченна стрічка), форми зворотного зв'язку без перезавантаження.

AJAX в jQuery — основні методи:

// Короткий запис для GET
$.get(url, function(дані) {
  $('#result').html(дані);
});

// Короткий запис для POST
$.post(url, { name: 'Оля' }, function(відповідь) {
  alert('Збережено!');
});

// Завантажити HTML у контейнер
$('#content').load('news.html #article');

// Повний контроль через $.ajax()
$.ajax({
  url:      '/api/data',
  method:   'GET',
  dataType: 'json',
  success:  function(data)  { відобразити(data); },
  error:    function(err)   { console.error(err); },
  complete: function()      { сховатиСпіннер();  }
});

🌐 AJAX — ЖИВИЙ ПРИКЛАД

// Отримати список користувачів з публічного API
$(function() {

  $('#load-btn').click(function() {

    // Показуємо індикатор завантаження
    $('#users-list').html('⏳ Завантаження...');

    // Відправляємо GET-запит
    $.get('https://jsonplaceholder.typicode.com/users',
      function(users) {

        // Будуємо HTML зі списку
        let html = '<ul>';
        users.slice(0, 5).forEach(function(user) {
          html += `<li>
            <b>${user.name}</b> — ${user.email}
          </li>`;
        });
        html += '</ul>';

        // Вставляємо у DOM
        $('#users-list').html(html);

      }
    ).fail(function() {
      $('#users-list').html('❌ Помилка завантаження');
    });

  });

});

🧩 JQUERY-ПЛАГІНИ — ГОТОВІ РІШЕННЯ

Що таке плагін jQuery?

Додатковий файл JavaScript, який розширює можливості jQuery. Підключається одним тегом <script> і миттєво дає нові методи.

💡 Як встановити додаток на телефон — одне натискання і нова функція готова.

Популярні плагіни:

  • 📅 jQuery UI — datepicker, draggable, sortable, dialog
  • 📊 DataTables — умні таблиці з сортуванням і фільтрами
  • 🖼️ Fancybox — лайтбокс для фотогалерей
  • 🎠 Slick — слайдери та каруселі
  • ✅ jQuery Validate — валідація форм
  • 📝 Select2 — красиві select-списки

jQuery UI — живий приклад:

📅 Datepicker (jQuery UI)

Код: $('#date').datepicker();

↕️ Sortable — перетягнути для сортування
  • 📦 Завдання A
  • 📌 Завдання B
  • 📋 Завдання C
  • 🎯 Завдання D

Код: $('#list').sortable();

⚖️ JQUERY, VANILLA JS ЧИ REACT — КОЛИ ЩО?

Критерій 🔷 jQuery ⚡ Vanilla JS ⚛️ React / Vue
Складність навчання ✅ Дуже проста 🟡 Середня ❌ Складна
Швидкість налаштування ✅ 1 рядок ✅ Нічого ❌ npm + Webpack
Продуктивність 🟡 Добра ✅ Відмінна ✅ Відмінна
Крос-браузерність ✅ Автоматично 🟡 Ручна ✅ Автоматично
Підходить для Існуючі сайти, лендінги, CMS (WordPress) Невелика інтерактивність, ігри Складні SPA, великі застосунки
Розмір 🟡 ~30 КБ ✅ 0 КБ ❌ 40–130 КБ

💡 Висновок: jQuery — ідеальний вибір для швидкого додавання інтерактивності на звичайний сайт, роботи з WordPress-темами та коли важлива простота. React/Vue — для великих бізнес-застосунків, де команда розробників працює місяцями.

🎓 Освіта

  • Прогрес-бари навчання

  • Форми запису на курси

🏥 Медицина та держпослуги

  • Фільтри пошуку лікарів / закладів

✅ ПІДСУМКИ ЛЕКЦІЇ

  • 🎯 jQuery — бібліотека JS, мета якої: «Write Less, Do More» — спростити роботу з вебсторінками
  • 💲 $() і CSS-селектори — серце jQuery; знайти елемент так само просто, як написати CSS-правило
  • 🔗 Chaining — ланцюговий виклик методів дозволяє писати лаконічний читабельний код
  • 🌳 DOM-маніпуляції — jQuery дозволяє легко читати, змінювати, додавати та видаляти елементи сторінки
  • 🖱️ Події — .click(), .hover(), .on() реагують на дії користувача одним рядком коду
  • ✨ Анімація — show/hide/fadeIn/slideDown та .animate() без CSS @keyframes
  • 🌐 AJAX — $.get(), $.post() дозволяють оновлювати частини сторінки без перезавантаження
  • 🧩 Плагіни — тисячі готових рішень (jQuery UI, DataTables, Slick тощо)

🏆 jQuery — найпопулярніша бібліотека в історії вебу! 🔷

📚 ДОМАШНЄ ЗАВДАННЯ

🎯 Базовий рівень (обов'язково):

  1. Підключіть jQuery до вашого HTML-файлу через CDN
  2. Зробіть кнопку, яка приховує/показує абзац через .slideToggle()
  3. Додайте лічильник символів до поля введення
  4. Динамічно додайте 3 елементи до списку через .append()

⭐ Підвищений рівень (за бажанням):

  1. Зробіть AJAX-запит до https://jsonplaceholder.typicode.com/posts і відобразіть перші 5 записів
  2. Підключіть jQuery UI datepicker до поля вводу дати
  3. Зробіть форму з валідацією (ім'я ≥ 2 символи, email містить @) без перезавантаження
