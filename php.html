💻 ПЕРШИЙ PHP-СКРИПТ

Будь-який PHP-код починається з <?php і закінчується ?>

📄 index.php

<?php
// Виводимо текст на сторінку
echo "Привіт, світ!";

// Поточна дата
$today = date("d.m.Y");
echo "Сьогодні: " . $today;
?>

🌐 Результат у браузері

Привіт, світ!
Сьогодні: 06.04.2026

📌 Браузер бачить лише готовий HTML.
PHP-код залишається на сервері — він невидимий для відвідувача.

📦 ЗМІННІ ТА ТИПИ ДАНИХ

У PHP змінна починається зі знаку $

<?php
// Рядок (текст)
$name = "Олена";

// Число
$age = 21;

// Дробове число
$price = 99.90;

// Логічний тип (так/ні)
$isStudent = true;

// Масив (список)
$subjects = ["Економіка", "Маркетинг", "PHP"];

echo "Студентка: " . $name;
echo "Вік: " . $age;
echo "Вивчає: " . $subjects[2];
?>

Типи даних PHP:

Тип Приклад
string (рядок) "Привіт"
integer (ціле) 42
float (дробове) 3.14
boolean (логіка) true / false
array (масив) [1, 2, 3]
null (нічого) null

<?php

🛠️ ФУНКЦІЇ — ГОТОВІ БЛОКИ КОДУ

Функція — це іменований блок коду, який можна викликати багато разів

<?php
// Оголошення функції
function greet($name) {
    return "Привіт, " . $name . "! 👋";
}

function discount($price, $percent) {
    $save = $price * $percent / 100;
    return $price - $save;
}

// Виклик функцій
echo greet("Олена");
echo greet("Іван");

$newPrice = discount(1000, 20);
echo "Ціна зі знижкою: " . $newPrice . " грн";
?>

🌐 Результат:

Привіт, Олена! 👋
Привіт, Іван! 👋
Ціна зі знижкою: 800 грн

💡 Чому це зручно?

✏️ Написав один раз — використовуй скрізь
🐛 Помилку виправив в одному місці — все працює
📚 PHP має тисячі вбудованих функцій (strlen, date, sort, array_map...)

🔗 PHP ВСЕРЕДИНІ HTML — ШАБЛОНИ

PHP вставляється прямо у HTML-розмітку — це і є «динамічна сторінка»

📄 profile.php

<!DOCTYPE html>
<html>
<body>

<?php
  $user = "Марія";
  $role = "Студентка";
  $year = 3;
?>

<h1>Вітаємо, <?= $user ?>!</h1>
<p>Роль: <?= $role ?></p>
<p>Курс: <?= $year ?>-й</p>

<?php if ($year >= 3): ?>
  <p>✅ Доступ до наукових статей</p>
<?php else: ?>
  <p>🔒 Лише базові матеріали</p>
<?php endif; ?>

</body>
</html>

🌐 Результат:

Вітаємо, Марія!

Роль: Студентка

Курс: 3-й

✅ Доступ до наукових статей


📌 Для кожного користувача сервер генерує свою унікальну HTML-сторінку.

🗄️ PHP + БАЗА ДАНИХ (MySQL)

PHP отримує, зберігає і оновлює дані у базі — серці будь-якого великого сайту

📰 Вивести новини з БД

<?php
// Підключення до бази даних
$db = new PDO(
  "mysql:host=localhost;dbname=site",
  "user", "password"
);

// SQL-запит: отримати 5 новин
$stmt = $db->query(
  "SELECT title, date FROM news
   ORDER BY date DESC LIMIT 5"
);
$news = $stmt->fetchAll();

// Виводимо кожну новину
foreach ($news as $row) {
    echo "<h3>" . $row['title'] . "</h3>";
    echo "<p>" . $row['date'] . "</p>";
}
?>

📬 КЕЙС 1: ФОРМА ЗВ'ЯЗКУ

Найпростіший реальний приклад — відправка email з сайту

<!-- HTML-форма (contact.html) -->
<form action="send.php" method="POST">
  <input name="email" type="email"
         placeholder="Ваш email">
  <textarea name="message"></textarea>
  <button type="submit">Надіслати</button>
</form>

<?php
// send.php — обробник форми
$email   = $_POST['email'];
$message = $_POST['message'];

mail(
  "admin@mysite.com",
  "Нове повідомлення",
  "Від: $email\n$message"
);

echo "✅ Повідомлення надіслано!";
?>

🔄 Як це працює:

1 Відвідувач заповнює форму та натискає «Надіслати»
2 Браузер відправляє дані на сервер (send.php)
3 PHP отримує дані через $_POST
4 PHP відправляє email адміністратору
5 Користувач бачить підтвердження ✅
