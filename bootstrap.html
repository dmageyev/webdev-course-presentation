Принципи та підходи

📐 ЩО ТАКЕ АДАПТИВНА ВЕРСТКА?

Адаптивна верстка (Responsive Web Design) — підхід до створення сайтів, при якому сторінка автоматично підлаштовує свій вигляд під розмір та можливості пристрою користувача.

❌ Без адаптивності

  • Сайт «зламаний» на телефоні
  • Потрібно горизонтально скролити
  • Текст занадто маленький
  • Кнопки не потрапляє пальцем

✅ З адаптивністю

  • Зручний вигляд на будь-якому екрані
  • Елементи перебудовуються автоматично
  • Великий текст і кнопки на телефоні
  • Оптимальний UX для кожного пристрою

🎯 ПРИНЦИП «MOBILE FIRST»

Mobile First — спочатку проектуємо для найменшого екрану, потім розширюємо для більших.

🔧 ТРИ ПРИНЦИПИ АДАПТИВНОСТІ

Засновник Ітан Маркотт (2010) виділив три ключові техніки:

🔲

Гнучка сітка
(Fluid Grid)

Розміри колонок задаються у відсотках, а не в пікселях. При звуженні вікна елементи пропорційно стискаються.

🖼️

Гнучкі медіа
(Flexible Media)

Зображення та відео автоматично масштабуються, щоб не виходити за межі контейнера.

📏

Медіа-запити
(Media Queries)

Правила стилів, що застосовуються лише при певних умовах: ширина екрану, орієнтація, тип пристрою.

🔲 ГНУЧКА СІТКА (FLUID GRID)

Розміри задаються відсотками від ширини батьківського елемента

BOOTSTRAP

Фреймворк для швидкої адаптивної верстки

🟣 ЩО ТАКЕ BOOTSTRAP?

Bootstrap — безкоштовний відкритий CSS-фреймворк з готовими стилями та компонентами для швидкого створення адаптивних вебсайтів.

📅
Створений у 2011

Розробники Twitter Mark Otto і Jacob Thornton відкрили для всіх

⭐
Найпопулярніший

170 000+ зірок на GitHub — один з найбільш завантажуваних CSS-фреймворків

🆓
Абсолютно безкоштовний

MIT-ліцензія — можна використовувати в будь-яких проєктах

⚡ ЧОМУ BOOTSTRAP? КЛЮЧОВІ ПЕРЕВАГИ

⚡
Швидкість розробки

Замість написання стилів «з нуля» — підключаєте готові рішення. Сайт готовий вдвічі-втричі швидше.

📱
Адаптивність «з коробки»

Вся система сітки Bootstrap спочатку адаптивна. Не потрібно писати медіа-запити вручну.

🧩
Готові компоненти

Кнопки, форми, навігація, картки, модальні вікна — все вже стилізовано та перевірено.

🌐
Крос-браузерність

Bootstrap перевірений у всіх популярних браузерах. Не треба виправляти помилки сумісності.

📚
Велика спільнота

Мільйони розробників, тисячі безкоштовних шаблонів, безліч підручників та відповідей.

🎨
Легка кастомізація

Колірна схема, шрифти, відступи — все гнучко налаштовується під ваш дизайн.

🛠️ УТИЛІТИ BOOTSTRAP

Окрім компонентів, Bootstrap надає сотні утилітарних класів — дрібних правил для швидкого налаштування

🎨 Кольори та фони

  • Колір тексту: синій, зелений, червоний, сірий…
  • Колір фону: будь-який з палітри Bootstrap
  • Однострочне написання без CSS-файлу

📐 Відступи та розміри

  • Зовнішні та внутрішні відступи: xs / sm / md / lg / xl
  • Ширина та висота у відсотках або фіксована
  • Маштабування під екран автоматично

👁️ Видимість

  • Приховати елемент на телефоні
  • Показати тільки на великому екрані
  • Адаптивне приховування без коду

🔤 Типографіка

  • Жирний, курсив, підкреслення
  • Вирівнювання: ліво, центр, право
  • Різні розміри тексту одним класом

🚀 ЯК ВИКОРИСТОВУВАТИ BOOTSTRAP?

Є кілька підходів — від простого до складного

🔗

1. CDN-підключення

Одне посилання у HTML-файлі — і Bootstrap готовий. Найпростіший спосіб для початківців та невеликих проєктів.

✅ Плюси: просто, швидко
⚠️ Мінуси: потрібен інтернет

🎨

2. Візуальні конструктори

Bootstrap Studio, Pinegrow та інші — drag&drop інтерфейс для складання сторінок без написання коду.

✅ Плюси: без коду
⚠️ Мінуси: платні інструменти

📄

3. Готові шаблони

Тисячі безкоштовних та платних HTML-шаблонів на Bootstrap — змінюєте лише текст і зображення.

✅ Плюси: максимально швидко
⚠️ Мінуси: потребує налаштування

✅ ПІДСУМКИ ЛЕКЦІЇ

1
Адаптивна верстка — сайт автоматично підлаштовується під будь-який екран: телефон, планшет, монітор.
2
Три принципи: гнучка сітка (відсотки), гнучкі медіа (масштабування), медіа-запити (умовні стилі).
3
Підхід Mobile First: спочатку проектуємо для телефону, потім розширюємо.
4
Bootstrap — безкоштовний фреймворк з готовою сіткою, компонентами та стилями для швидкої адаптивної верстки.
5
Використовуйте CDN-підключення, готові шаблони або візуальні конструктори — вибір залежить від задачі.
6
Bootstrap використовує ~22% сайтів у світі — це стандарт галузі, що варто знати кожному, хто працює з вебом.

🟣 Завдання: Лендінг-сторінка на Bootstrap

Використовуючи готовий Bootstrap-шаблон або конструктор, створіть просту лендінг-сторінку:

✅ Обов'язкові елементи

  • Навігаційна панель з назвою
  • Секція-hero із заголовком та кнопкою
  • Блок із 3 картками (переваги/послуги)
  • Підвал (footer)

⭐ Додаткові бали

  • Коректне відображення на телефоні
  • Форма зворотного зв'язку
  • Карусель зображень
  • Власна колірна схема
