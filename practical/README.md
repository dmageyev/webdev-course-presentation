## 📚 Теоретична частина

### 2.1 Що таке CSS?

**CSS** (Cascading Style Sheets — каскадні таблиці стилів) — це мова для **оформлення** HTML-документів.

CSS відповідає за кольори, шрифти, розміри, відступи, анімації та все, що робить сторінку красивою.

```
HTML = СТРУКТУРА сторінки
CSS  = ЗОВНІШНІЙ ВИГЛЯД сторінки
```

> 💡 **Аналогія:** HTML — це кістяк будинку (стіни, кімнати), а CSS — це ремонт (фарба, меблі, декор).

---

### 2.2 Синтаксис CSS-правила

```
селектор {
  властивість: значення;
  властивість: значення;
}
```

**Приклад:**

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

| Частина правила | Що означає              | Де знаходиться |
|-----------------|-------------------------|----------------|
| `h1`            | Селектор (кого стилізуємо) | Перед `{`   |
| `color`         | Властивість (що змінюємо)  | Всередині `{}` |
| `blue`          | Значення (на що змінюємо)  | Після `:`   |
| `;`             | Крапка з комою — кінець декларації | Після значення |

> ⚠️ **Важливо:** Кожна декларація закінчується крапкою з комою `;`

---

### 2.3 Способи підключення CSS

Існує 3 способи додати CSS до HTML-сторінки:

#### Спосіб 1 — Вбудований стиль (inline)
```html
<h1 style="color: red;">Текст</h1>
```
⚠️ Лише для одного елемента. Важко підтримувати.

#### Спосіб 2 — Внутрішній тег `<style>`
```html
<head>
  <style>
    h1 { color: red; }
  </style>
</head>
```
✅ Підходить для невеликих проєктів.

#### Спосіб 3 — Зовнішній файл ⭐ (рекомендовано)
```html
<head>
  <link rel="stylesheet" href="style.css">
</head>
```
⭐ **Найкращий спосіб!** Стилі відокремлені від HTML, легко підтримувати.

---

### 2.4 Основні CSS-властивості

| Властивість        | Що робить                | Приклад                               |
|--------------------|--------------------------|---------------------------------------|
| `color`            | Колір тексту             | `color: blue;`                        |
| `background-color` | Колір фону               | `background-color: #fff;`             |
| `font-size`        | Розмір шрифту            | `font-size: 18px;`                    |
| `font-weight`      | Жирність                 | `font-weight: bold;`                  |
| `font-family`      | Шрифт                    | `font-family: Arial;`                 |
| `text-align`       | Вирівнювання тексту      | `text-align: center;`                 |
| `border`           | Рамка                    | `border: 2px solid red;`              |
| `border-radius`    | Закруглення кутів        | `border-radius: 10px;`                |
| `padding`          | Внутрішній відступ       | `padding: 20px;`                      |
| `margin`           | Зовнішній відступ        | `margin: 10px auto;`                  |
| `width` / `height` | Ширина / Висота          | `width: 300px;`                       |
| `box-shadow`       | Тінь блоку               | `box-shadow: 0 4px 10px rgba(0,0,0,0.2);` |
| `transition`       | Плавний перехід (hover)  | `transition: all 0.3s ease;`          |
| `animation`        | CSS-анімація             | `animation: spin 2s infinite;`        |

## 🔷 Блокові схеми

### Схема 1: Як браузер будує веб-сторінку

```
 ┌──────────┐      ┌──────────┐      ┌──────────┐      ┌──────────────┐
 │ 📝 HTML  │ ───▶ │ 🎨 CSS   │ ───▶ │ 🌐       │ ───▶ │ ✨ Веб-     │
 │  файл    │      │  файл    │      │ Браузер  │      │  сторінка    │
 │          │      │          │      │          │      │              │
 │Структура │      │  Стилі   │      │ Читає та │      │  Красивий    │
 │сторінки  │      │оформлення│      │ поєднує  │      │  результат!  │
 └──────────┘      └──────────┘      └──────────┘      └──────────────┘
```

---

### Схема 2: CSS Блокова модель (Box Model)

Кожен HTML-елемент — це прямокутний блок із чотирьох шарів:

```
╔══════════════════════════════════════╗
║          MARGIN (зовнішній відступ)  ║
║  ╔══════════════════════════════╗    ║
║  ║    BORDER (рамка)            ║    ║
║  ║  ╔══════════════════════╗    ║    ║
║  ║  ║  PADDING (внутр. відступ) ║    ║
║  ║  ║  ╔════════════════╗  ║    ║    ║
║  ║  ║  ║    CONTENT     ║  ║    ║    ║
║  ║  ║  ║   (контент)    ║  ║    ║    ║
║  ║  ║  ╚════════════════╝  ║    ║    ║
║  ║  ╚══════════════════════╝    ║    ║
║  ╚══════════════════════════════╝    ║
╚══════════════════════════════════════╝
```

| Шар       | CSS-властивість | Що це?                            |
|-----------|-----------------|-----------------------------------|
| `content` | `width`/`height`| Сам вміст елемента (текст, фото)  |
| `padding` | `padding`       | Відступ між вмістом і рамкою      |
| `border`  | `border`        | Рамка навколо елемента            |
| `margin`  | `margin`        | Відступ між цим і сусіднім елементами |

---

### Схема 3: Типи CSS-селекторів

```
┌─────────────────┬────────────────────┬────────────────────────────────┐
│  Тип            │  Синтаксис         │  Що вибирає                    │
├─────────────────┼────────────────────┼────────────────────────────────┤
│ Тег             │  h1 { }            │ Всі теги <h1> на сторінці      │
│ Клас            │  .red { }          │ Елементи з class="red"         │
│ Ідентифікатор   │  #header { }       │ Один елемент з id="header"     │
│ Псевдоклас      │  a:hover { }       │ Стан при наведенні миші        │
└─────────────────┴────────────────────┴────────────────────────────────┘
```

**Приклад використання:**

```html
<!-- HTML -->
<h1>Назва</h1>               <!-- тег -->
<p class="red">Текст</p>     <!-- клас -->
<div id="header">Шапка</div> <!-- id -->
<a href="#">Посилання</a>    <!-- псевдоклас :hover -->
```

```css
/* CSS */
h1        { color: blue; }          /* тег-селектор     */
.red      { color: red; }           /* клас-селектор    */
#header   { background: gray; }     /* id-селектор      */
a:hover   { text-decoration: none; }/* псевдоклас       */
```

### Крок 3 — Додай стилі для `body`

Почни з загальних стилів усієї сторінки:

```css
body {
  background-color: #f0f8ff;
  font-family: Arial, sans-serif;
  padding: 20px;
}
```

**Що тут відбувається:**
- `background-color: #f0f8ff` — блакитний фон сторінки
- `font-family: Arial, sans-serif` — шрифт для всього тексту
- `padding: 20px` — відступ від країв сторінки

---

### Крок 4 — Стилізуй заголовки

```css
h1 {
  color: #4285F4;
  text-align: center;
  border-bottom: 3px solid #4285F4;
  padding-bottom: 10px;
}

h2 {
  color: #34A853;
  margin-top: 20px;
}
```

---

### Крок 5 — Стилізуй параграфи

```css
p {
  background-color: white;
  padding: 15px;
  border-radius: 10px;
  border-left: 5px solid #34A853;
  margin: 10px 0;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}
```

**Що тут відбувається:**
- `background-color: white` — білий фон для параграфа
- `padding: 15px` — відступ всередині
- `border-radius: 10px` — закруглені кути
- `border-left: 5px solid #34A853` — зелена ліва рамка
- `box-shadow` — тінь

---

### Крок 6 — Додай ефект при наведенні `:hover`

```css
h1:hover {
  color: #EA4335;
  transition: color 0.3s ease;
}

p:hover {
  transform: translateX(5px);
  transition: transform 0.3s ease;
}
```

> 💡 `transition` робить перехід **плавним** — без нього зміна відбувається миттєво.

---

### Крок 7 — Додай CSS-анімацію (`@keyframes`)

```css
h2 {
  animation: slideIn 0.5s ease;
}

@keyframes slideIn {
  from {
    transform: translateX(-40px);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
```

**Як читати `@keyframes`:**
- `from` — початковий стан (зліва, прозорий)
- `to` — кінцевий стан (на місці, видимий)
- `0.5s` — тривалість 0.5 секунди

## 📖 Шпаргалка CSS

### Кольори

```css
color: red;                    /* ключове слово */
color: #EA4335;                /* HEX-код */
color: rgb(234, 67, 53);       /* RGB */
color: rgba(234, 67, 53, 0.5); /* RGB + прозорість */
```

### Шрифти

```css
font-size: 16px;       /* розмір у пікселях */
font-size: 1.2em;      /* відносний розмір */
font-weight: bold;     /* жирний */
font-weight: 700;      /* теж жирний (число 100–900) */
font-style: italic;    /* курсив */
font-family: Arial, sans-serif;
```

### Відступи та розміри

```css
padding: 20px;              /* з усіх боків */
padding: 10px 20px;         /* вертикальні | горизонтальні */
padding: 10px 20px 15px 5px;/* зверху | справа | знизу | зліва */

margin: 0 auto;             /* по центру горизонтально */

width: 300px;
max-width: 700px;
height: 150px;
```

### Рамки та фон

```css
border: 2px solid #4285F4;        /* товщина стиль колір */
border-left: 5px solid green;     /* тільки ліва */
border-radius: 10px;              /* закруглення */
border-radius: 50%;               /* коло */

background-color: #f0f8ff;
background: linear-gradient(135deg, #4285F4, #9C27B0);
box-shadow: 0 4px 10px rgba(0, 0, 0, 0.2);
```

### Текст

```css
text-align: center;    /* вирівнювання: left | center | right */
text-decoration: none; /* без підкреслення (для посилань) */
text-transform: uppercase; /* ВЕЛИКІ ЛІТЕРИ */
line-height: 1.6;      /* міжрядковий інтервал */
```

### Ефекти та анімації

```css
/* Плавний перехід для hover */
transition: all 0.3s ease;
transition: color 0.5s ease-in-out;

/* Трансформації */
transform: translateY(-10px);  /* рух вгору */
transform: scale(1.05);        /* масштаб */
transform: rotate(5deg);       /* поворот */

/* Анімація */
animation: myAnim 1s ease infinite;

@keyframes myAnim {
  from { opacity: 0; }
  to   { opacity: 1; }
}
```
