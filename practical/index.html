📚 Теоретична частина

2.1 Що таке CSS?

CSS (Cascading Style Sheets — каскадні таблиці стилів) — це мова для оформлення HTML-документів.

CSS відповідає за кольори, шрифти, розміри, відступи, анімації та все, що робить сторінку красивою.

HTML = структура сторінки  |  CSS = зовнішній вигляд сторінки.

2.2 Синтаксис CSS-правила

h1 {
color : blue ;
font-size : 24px ;
}

2.3 Способи підключення CSS

1

Вбудований стиль

<h1 style="color: red;">
  Текст
</h1>

⚠️ Лише для одного елемента

2

Внутрішній <style>

<style>
  h1 { color: red; }
</style>

✅ Для невеликих проєктів

3

Зовнішній файл ⭐

<link rel="stylesheet"
  href="style.css">

🎨 "Моя стилізована сторінка"

Візьми свою HTML-візитку з попереднього заняття і зроби її красивою за допомогою CSS!

⏱️ Час: 20–30 хвилин 🛠️ Тільки HTML + CSS 📍 Платформа: CodePen

✅ Мінімальні вимоги

  • Змінити колір тексту (color)
  • Додати фоновий колір сторінки (background-color)
  • Змінити шрифт або розмір тексту
  • Додати рамку до одного блоку (border)
  • Вирівняти заголовок по центру
  • Додати відступи (padding)

⭐ Бонусні вимоги

  • Hover-ефект на заголовок або кнопку
  • CSS transition для плавності
  • Хоча б одна @keyframes анімація
  • Тінь блоку (box-shadow)
  • Градієнтний фон (linear-gradient)
  • Закруглені кути (border-radius)

🖊️ Приклад CSS-коду (вкладка CSS у CodePen):

/* Загальні стилі сторінки */
body {
  background-color: #f0f8ff;
  font-family: Arial, sans-serif;
  padding: 20px;
}

/* Головний заголовок */
h1 {
  color: #4285F4;
  text-align: center;
  border-bottom: 3px solid #4285F4;
  padding-bottom: 10px;
  transition: color 0.3s ease;
}

/* Ефект при наведенні на h1 */
h1:hover {
  color: #EA4335;
}

/* Розділи */
h2 {
  color: #34A853;
  animation: slideIn 0.5s ease;
}

/* Анімація для h2 */
@keyframes slideIn {
  from {
    transform: translateX(-40px);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

/* Параграфи */
p {
  background-color: white;
  padding: 15px;
  border-radius: 10px;
  border-left: 5px solid #34A853;
  margin: 10px 0;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}
